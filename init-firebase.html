Configuration Firebase

Activez l'authentification pour accéder au panel admin

📋 Étape 1 : Activer Email/Password

La méthode de connexion par email/mot de passe n'est pas encore activée dans votre projet Firebase.

Ouvrir Firebase Console

⚙️ Étape 2 : Instructions

Une fois sur la page Firebase Console :

  1. Cliquez sur "Authentication" dans le menu de gauche
  2. Allez dans l'onglet "Sign-in methods"
  3. Repérez "Email/Password"
  4. Cliquez sur "Activer" (toggle button)
  5. Cliquez sur "Enregistrer"

🔑 Étape 3 : Créer un compte admin

Après avoir activé Email/Password, créez un compte administrateur :

Gérer les utilisateurs

🗄️ Étape 4 : Activer Firestore

Activer Firestore
